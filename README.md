# AulasThreadsCourotines

# Threads, Coroutines & API (Kotlin Android)

## About the Project

This repository contains practical Android development studies focused on:   
Threads   
Kotlin Coroutines   
API consumption with Retrofit   
GET and POST requests   
Background tasks   
UI interaction with asynchronous operations   
The project was built during Android development lessons to practice how applications handle network requests and concurrent tasks without blocking the UI.  

## Technologies Used   

Kotlin   
Android Studio   
Retrofit   
Coroutines   
REST API   
ViewBinding   
XML Layouts   
Threads   
AppCompat   

## Features Practiced   

✅ API requests with Retrofit   
✅ GET requests   
✅ POST requests   
✅ Handling comments and posts   
✅ Coroutines with suspend functions   
✅ Background processing   
✅ UI updates safely on Main Thread   
✅ Thread control (start/stop)   
✅ Navigation between screens   

## What I Learned     

During this project I practiced important Android concepts such as:   
Difference between Threads and Coroutines   
How asynchronous tasks work   
How to avoid blocking the UI thread   
API integration using Retrofit   
Organizing code into packages and models   
Using coroutines for cleaner and safer asynchronous code
